git
1.第一次下载 git clone 仓库地址
2.纳入版本控制 git add .
2.提交到本地 git commit -m “简单描述信息”
4.推送到远程仓库 git push origin master
5，拉取 git pull 仓库地址/origin
6.查看状态 git status
7.查看日志 git log 
8.删除远程拉去最新  git checkout master  → git reset --hard
